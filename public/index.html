Shape Game Details
  • Goal: Build a shape-drawing app with my main focus being on practicing front-end design.
  • Important Lesson I Learned: How even a “simple app” can be rather complex. This project helped solidify the concept of JS class constructors for me, and I finally understood that constructors are necessary to instantiate new objects (with necessary properties) created by a class. In addition, I learned how important my junior high math skills can be for coding: they helped me resolve two issues. First, how do I draw a triangles? (It was proving to be incredibly tricky). For this, I used the border of an empty square shape and colored one corner to draw triangles. Who knew that was so complicated? And second, how do I keep shapes from drawing outside of the canvas? For this, I calculated the max amount of pixels that each shape could contain to keep it limited to the area within the canvas’s perimeter.
  • New Technologies/Skills Used: Object Oriented Programming and inheritance; lots of math

Typing Game Details
  • Goal: I was very excited to get this game working (I was a Mavis Beacon ace as a kid), but I knew the mechanics of it (i.e. the jQuery) would require extra attention. My primary aim was to make the game function well, have a good UI even if it wasn’t the prettiest, and to add proper game instructions to improve the UX.
  • Important Lesson I Learned: How to think a bit outside of the box in jQuery as a novice developer. This game required using keyCodes and event.which in order for the app to identify which keys were being pressed during gameplay; therefore, I had to learn that unfamiliar ways of selecting elements, such as $(‘#’ + e.which), were valid and really useful. After that, my last challenge was placing the yellow block/highlighter directly over the correct letter that was to be pressed on the keyboard. I made countless tiny pixel tweaks to position it correctly, but it didn’t sit perfectly centered over each letter until I finally tried the addition assignment operator (+=).
  • New Technologies/Skills Used: JS keyCodes, keypress events, and event.which; bootstrap modals
